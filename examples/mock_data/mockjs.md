# MockJS

- Data Template Definition/DTD
- Data Placeholder Definition/DPD

## DTD

```
name|rule`:value
```

rule:

```
'name|min-max': value
'name|count': value
'name|min-max.dmin-dmax': value
'name|min-max.dcount': value
'name|count.dmin-dmax': value
'name|count.dcount': value
'name|+step': value
```
